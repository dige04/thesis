\subsection{Full Memory}

Full Memory is the maximum-dose ceiling. Every completed task appends a record and none is
ever evicted: the cap of ten (A1) is ignored by design, so the store grows monotonically
over the sequence. This makes Full Memory the upper bound on memory \emph{footprint} --- it
carries the largest store any condition reaches and anchors the heavy end of the footprint
axis --- and it is the accumulation strategy the rest of the field defaults to. It is
therefore the natural comparison target: proactive forgetting succeeds if it matches Full
Memory on task resolution (the non-inferiority hypothesis, H1) while carrying a smaller
footprint (H1b).
